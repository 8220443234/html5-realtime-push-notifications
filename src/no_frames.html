PHP HTML5 Push Notifications using Pusher

PHP HTML5 Push Notifications

This page demonstrates HTML5 Realtime Push Notifications using Pusher. This functionality can easily be added to any PHP application.

The demos below show how to:

  • Trigger CRUD (Create, Read, Update & Delete) notifications
  • Trigger an announcement notification
  • Trigger events using traditional HTTP actions

All demos use a PHP Pusher library and a helper library which is included within the source of the demos.

CRUD notifications Trigger Create, Read, Update and Delete notifications

Create, Read, Update & Delete are the four basic functions of persistent storage. They also indicate actions and therefore events. Notifications are used to indicate that an event has occurred so CRUD actions are the most obvious use of Realtime Push Notifications.

Simply click a button to trigger the appropriate CRUD notification.


Announcements Trigger an announcement notification

Just enter some text into the textarea and click the announce button to trigger an announcement containing the text you've entered.


Actions Trigger events using traditional HTTP actions

The iframe is your view into the standard world of HTTP. Simple navigate by clicking the links and notifications will be triggered. The source is available for each page: create = create.php, read = read.php, update = update.php and delete = delete.php.
